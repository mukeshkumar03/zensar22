# Frontend (Angular) Placeholder

This folder is a placeholder for the Angular frontend. To scaffold an Angular project here, run:

```bash
cd frontend
npx @angular/cli new order-management-frontend --routing --style=scss
```

Create components:

```bash
cd order-management-frontend
ng generate component orders
ng generate component customers
ng generate component payments
```

To run:

```bash
cd order-management-frontend
npm install
ng serve --open
```
